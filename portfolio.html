Sentiment Analysis of the Boycott Movement on X #Boycott Using Support Vector Machine

This research is an implementation of artificial intelligence through a machine learning approach with the SVM algorithm, which the entire process is carried out using the Python programming language. Data was obtained by crawling the #Boycott product hashtag during January-April 2025, then processed with text preprocessing techniques, TF-IDF weighting, and data balancing using SMOTE. SVM models were tested with two kernels, namely linear and RBF.

UML System Design – BISON

BISON system design uses the UML (Unified Modeling Language) approach to model the flow and structure of the system in a structured and easy to understand manner. The diagrams used include Use Case Diagram to describe the interaction between actors such as passengers, bus operators, and admins with the system; Activity Diagram to show the flow of ticket booking from login to e-ticket; Class Diagram that models entities such as users, buses, routes, tickets, and transactions; Sequence Diagram to explain the sequence of interactions between objects in processes such as payment; and Deployment Diagram that describes the physical architecture of the system. The application of UML helps the development of the BISON system to be more focused and efficient, as well as facilitating the understanding of system requirements and testing.
